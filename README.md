# Objetivo do Projeto

O objetivo deste trabalho foi desenvolver 18 exercícios utilizando TypeScript, organizados dentro de um único sistema interativo em terminal.

Ao invés de criar vários arquivos separados, o grupo optou por desenvolver um HUB principal com menu interativo, permitindo ao usuário selecionar e executar cada exercício individualmente.

Essa abordagem tornou o projeto:
- mais organizado;
- mais fácil de executar;
- mais fácil de apresentar;
- mais fácil de corrigir;
- mais profissional visualmente.

---

1. escolhe um exercício;
2. digita os dados solicitados;
3. visualiza o resultado;
4. pode voltar ao menu principal para executar outro exercício.
